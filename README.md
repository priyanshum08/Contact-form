# Contact Form Management System

A full-stack Contact Form Management System developed using **HTML, CSS, JavaScript, PHP, and MySQL**.  
This project provides a responsive contact interface for users along with an admin management system to handle submitted messages efficiently.

The application is designed as a college-level web technology project demonstrating frontend development, backend integration, database connectivity, and basic admin-side operations using PHP and MySQL.

---

## Project Overview

The system allows users to submit contact information and messages through a modern responsive form.  
Submitted records are processed using PHP and stored in a MySQL database.  
An admin module is included for managing and viewing user submissions.

This project demonstrates:

- Frontend and backend integration
- PHP form handling
- MySQL database operations
- Responsive web design
- Basic admin dashboard functionality
- CRUD-based workflow concepts

## Installation and Setup

### Step 1 — Clone the Repository

```bash
git clone https://github.com/priyanshum08/Contact-form.git
```

---

### Step 2 — Move Project to XAMPP htdocs

Move the project folder to:

```bash
/Applications/XAMPP/xamppfiles/htdocs/
```

Final directory:

```bash
/Applications/XAMPP/xamppfiles/htdocs/contact-form-project
```

---

### Step 3 — Start XAMPP Services

Open XAMPP and start:

- Apache
- MySQL

---

### Step 4 — Import Database

1. Open phpMyAdmin:

```bash
http://localhost/phpmyadmin
```

2. Create a new database.

3. Import the SQL file available inside the `database` folder.

---

### Step 5 — Run the Project

Open your browser and visit:

```bash
http://localhost/contact-form-project
```
